# 🧾 Inventory Management System

A backend API built using **Node.js**, **Express**, **MongoDB**, and **Mongoose** to manage an inventory system for a shopping cart-style web application. It includes product CRUD operations, JWT-based authentication (optional), and supports features like low stock alerts and restocking.

---

## 🚀 Features

- 🔐 **User Authentication** (with JWT)
- 📦 **Product Management (CRUD)**
- ⚠️ **Low Stock Alerts** (optional)
- 🔁 **Restocking Support**
- 🕓 **Timestamps for Created/Updated Products**
- 📦 Uses **MongoDB** for NoSQL database storage
- 📡 Built with **RESTful API design principles**

---

## 🛠️ Technologies Used

- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **dotenv** (for environment config)
- **Postman** (for testing)
